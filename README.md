# What's that?
That's my project named [Lightfetch](https://github.com/wcupped/lightfetch) that is rewritten on Rust. That's actually a hobby project for me and is not going to be seriously supported as a Lightfetch itself.
# Installation
You can install through cargo, entering `cargo install rightfetch` in your terminal, alternatively, compiling it yourself and then dropping the binary file in `/usr/bin` path.
